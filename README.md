#  OwlMath — Мобильное приложение для изучения математики

Добро пожаловать в **OwlMath**! Это яркое, интуитивно понятное и геймифицированное мобильное приложение для изучения математики, разработанное на базе **React Native** и **Expo v56**. Дизайн интерфейса вдохновлен философией Duolingo и выполнен в премиальных фиолетовых тонах с элементами микроанимаций, кастомными тенями и поддержкой различных уровней сложности.


---

##  Дизайн и Дизайн-система

Приложение OwlMath спроектировано с акцентом на визуальное превосходство и вовлечение пользователя:
*   **Гармоничная палитра (Duolingo-inspired):** Основной фиолетовый бренд-цвет (`#7C3AED`), золотые акценты для серий занятий и XP (`#F59E0B`), мягкие градиенты, чистый белый фон поверхностей и контрастный текст (`#1E1B4B`).
*   **Игровая геймификация:**
    *    **Система жизней:** 5 сердечек на старте, которые тратятся при неверных ответах.
    *    **Серии дней (Streaks):** Трекинг ежедневной активности для мотивации регулярных занятий.
    *    **Уровни и XP:** Опыт начисляется за правильные ответы с возможностью повышения общего уровня (до 10 уровней с динамической шкалой прогресса).
*   **Интерактивность:** Гладкие переходы, выразительные смайлики-иконки (📚, 🏆, 🦉, ❤️, 🔥) и отзывчивый интерфейс с поддержкой тактильного отклика на действия пользователя.

---

##  Ключевые возможности приложения

1.  ** Полноценная авторизация и сессии:**
    *   Экран входа (`LoginScreen`) и регистрации (`RegisterScreen`) с полной валидацией полей (длина имени, формат email, надежность пароля).
    *   Сохранение и автоматическое восстановление текущего пользователя через `AsyncStorage`.
    *   Безопасный логаут с очисткой сессии в профиле.
2.  ** Интерактивные математические викторины:**
    *   Генерация вопросов на лету из локальной базы данных задач в формате CSV (`math_tasks_with_answers.csv`).
    *   Три градации сложности:
        *   **Начальная школа ** — арифметика, простые текстовые задачи, площади и периметры.
        *   **Средняя школа ** — уравнения, дроби, проценты, базовые степени, геометрия и НОД/НОК.
        *   **Старшая школа ** — квадратные уравнения, логарифмы, тригонометрия, производные и интегралы, теория вероятностей, стереометрия.
3.  ** Динамическая таблица лидеров (`LeaderboardScreen`):**
    *   Рейтинг учеников на основе их общего опыта (XP) и правильных ответов.
    *   Визуальные маркеры топ-3 призеров.
4.  ** Интерактивный профиль (`ProfileScreen`):**
    *   Персонализированная аватарка совы.
    *   Отображение текущей серии дней (Streak), баланса XP и шкалы прогресса до следующего уровня.
    *   История выполненных уроков по каждой из категорий.
    *   Возможность сбросить весь прогресс для повторного прохождения.

## Формат базы данных математических задач (CSV)

Все задачи хранятся в файле `math_tasks_with_answers.csv` и структурированы следующим образом:

| Колонка | Описание | Пример значения |
| :--- | :--- | :--- |
| `Global_ID` | Уникальный глобальный идентификатор задачи (1-150) | `109` |
| `Level_ID` | Идентификатор задачи внутри уровня сложности (1-50) | `9` |
| `Level` | Уровень сложности (`Начальная школа`, `Средняя школа`, `Старшая школа`) | `Старшая школа` |
| `Task` | Текст вопроса или математическое выражение | `Вычислите: log_2(32)` |
| `Answer` | Строковый эталонный ответ для автоматической проверки | `5` |

##  Установка и запуск

Для запуска приложения на локальном компьютере вам понадобятся установленные **Node.js** (версии 18+) и менеджер пакетов **npm**.

### Шаг 1: Установка зависимостей
Установите все необходимые npm-пакеты:
```bash
npm install
```

### Шаг 2: Запуск сервера разработки
Запустите Expo Metro Bundler:
```bash
npm run start
```

После запуска в терминале отобразится QR-код. 

*   **Для тестирования на мобильном телефоне (iOS/Android):** Установите приложение **Expo Go** из App Store или Google Play, откройте камеру (на iOS) или приложение Expo Go (на Android) и отсканируйте этот QR-код. Ваше мобильное устройство должно находиться в той же локальной сети Wi-Fi, что и компьютер.
*   **Для запуска на Web:** Нажмите клавишу `w` в консоли или выполните команду `npm run web`.
*   **Для запуска на эмуляторах:** Нажмите `a` для запуска на Android-эмуляторе или `i` для iOS-симулятора.

## Стек технологий

*   **Фреймворк:** React Native & Expo (v56.0.4)
*   **Язык программирования:** JavaScript (React 19, React Native 0.85)
*   **Навигация:** `@react-navigation/native` (Bottom Tabs Navigation, Stack Navigation)
*   **Постоянное хранение:** `@react-native-async-storage/async-storage`
*   **Анимации и стилизация:** Ванильный React Native StyleSheet с кастомными адаптивными токенами.
*   **Интерфейсы:** Поддержка векторной графики (`react-native-svg`), анимации переходов экранов (`react-native-screens`).
